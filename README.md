# 命运星盘 - AI星座占卜小程序

> 基于Taro + React开发的微信小程序，集成AI智能解读功能

## 功能模块

- **MBTI人格测试** - 16型人格分析，AI深度解读
- **塔罗牌占卜** - 三牌阵（过去/现在/未来），每日免费抽取
- **五行八字** - 生辰八字测算，五行分布分析

## 技术栈

- Taro 4.x + React
- TypeScript
- MiniMax AI API（待接入）
- 微信云开发（待接入）

## 开发

```bash
npm install
npm run dev:weapp    # 开发模式
npm run build:weapp # 构建生产版本
```
